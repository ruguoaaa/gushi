作为职业交易者，复盘是每日必修课。以下是我15年实战总结的高效复盘流程（浓缩版）：

### 一、复盘核心框架（30分钟完成）

1. **市场温度计（3分钟）**

- 涨跌停家数比（同花顺输入"60"）
- 昨日涨停今表现（通达信输入".403"）
- 炸板率/连板晋级率（手动记录）

*案例：2023-08-28*
涨停：26家 → 情绪冰点
炸板率：48% → 接力风险极高
连板晋级：3进4全灭 → 回避高位

2. **资金流向（5分钟）**

- 行业板块资金TOP5（东方财富板块资金流）
- 龙虎榜机构动向（重点看净买超5000万标的）
- 北向资金重点加仓（同花顺北向持股）

*工具：大智慧L2逐笔委托回放关键个股分时*

3. **异动暗线（7分钟）**

- 涨幅≥7%非涨停股（挖掘潜在补涨）
- 尾盘抢筹TOP10（通达信".401"按涨幅排序）
- 振幅前20个股（发现分歧转一致机会）

4. **周期定位（10分钟）**

- 连板高度演变图（手工绘制情绪周期）
- 主流题材持续性（华为链/机器人/光伏等）
- 新旧题材卡位点（如CPT取代存储芯片）

5. **交易计划（5分钟）**

- 持仓标的处理预案
- 备选攻击目标池
- 风险预警信号清单

### 二、顶级游资独门技巧

1. **涨停基因筛查法**

- 历史股性检测（近1年涨停次日后3日平均收益）
- 隐形关联度分析（同实控人/业务协同）
- 机构研报真空期（避开强逻辑证伪期）

*案例：2023-07-11中马传动*

- 历史8次涨停后平均溢价3.2%
- 实控人旗下另一公司当日异动
- 近3月无券商覆盖

2. **集合竞价博弈术**

- 隔夜单撤单比例（9:15-9:25观测）
- 匹配量/流通盘比值（＞0.3%为强势）
- 关键档位托单（买一/卖一厚度比）

3. **盘口语言破译**

- 万手单拆解（L2数据逐笔还原）
- 压单/托单组合（如4444手压单示警）
- 涨停板排单结构（机构通道识别）

### 四、复盘工具配置清单

- 情绪监控：开盘啦APP（打板热力图）
- 龙虎榜分析：选股宝（席位联动追踪）
- 产业链图谱：理杏仁（华为供应链穿透）
- 资金监控：同花顺资金博弈指标
- 自建模型：Python涨停因子回测系统

### 五、顶级游资忠告

1. **信息降噪**：过滤90%的无效消息，只跟踪顶级游资动向
2. **反共识交易**：当论坛热帖讨论度超70%时警惕见顶
3. **仓位控制**：单日回撤＞3%立即启动熔断机制
4. **交割单审计**：每周统计胜率/赔率，砍掉负EV策略

*最后提醒：上述方法需配合严格的交易纪律，新手建议先用模拟盘验证3个月再实盘操作。*
